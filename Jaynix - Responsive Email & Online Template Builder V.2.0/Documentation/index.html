Hello there!

Thank you for purchasing our product.
This Documentation will explain you how to work with our Email Templates.

Replace Images

To replace the images open the folder called "images" and replace them with your images. You can see the width and height of our images by clicking on "information".

HTML Code

Open the HTML file in a code editor to modify the code or replace URL links and/or images in the code.

PSD File

Open the PSD file in Adobe Photoshop to modify the design.

Online Template Builder

The following steps will show you how to work with our Online Template Builder.

Login
Go to http://www.builder.dynamicxx.com/login and login to get access to the Builder.


Register
If you don't have an account yet, please go to http://www.builder.dynamicxx.com/register and register to get access to the Builder.
Once you've registered you're able to login and start using your template in the Builder.

Unlock template / Bundle
Press the Unlock template / bundle button.

In the download folder there's folder called "Purchase Code.txt". Open this and copy the code.


Once you've copied the code, paste it in the input field and press the Unlock button.


Modules
Drag & Drop the modules into the Builder to use them.

Premade
- Use the Premade modules (full width image, 2 col, button etc) to build an email template the way you like it.

Preview
Press the "Preview" tab to see how the Template looks on Desktop, Tablet or Mobile.


Send tests
Send tests to see how the email looks in your inbox.
Remember that images which are uploaded from desktop are not stored on the server and those images won't be show up in the inbox unless you upload images by URL links. Don't worry, once the template will be send though an ESP your image will show up in the inbox as the image will be stored on the platform before sending the out the email.


Export your Template
Click on the Export tab and press "MailChimp", "Campaign Monitor", or "HTML Normal" to export your file to desktop.


Drag & Drop Modules
Press the 1st button to Drag&Drop Modules.


Duplicate Modules
Press the 2nd button to duplicate a module.


Color / Background Editor
Press the 3rd button to change colors or background images. Our Builder detect colors and background images automatically which you can edit.

Once you've pressed the button, an editor collapse where you can edit background colors or background images. You can upload background images by entering a valid URL or upload from desktop.

Code Editor
Press the 4th button to open the code editor. Please be sure you have understanding of HTML / CSS before changing the code as it might break the template.

Once you've pressed the button, the code editor collapse where you can edit source.

Delete Modules
Press the 5th button remove modules.


Undo / Redo
In the top bar there's an Undo and Redo button to undo and redo custom work in the Builder.


Text Editor
Press any text in the Builder to edit the text.


Change Images
Press an image to change the image. You can upload by desktop or enter a valid URL. You can also add links on images or completely remove them by pressing the "Delete image" button.


Reset
Press the "Reset" button in the top bar to reset ALL changes.


Rename templates
Enter a name and press the "Rename" button to change the name of your template.

Once you've renamed a template it will automatically saved in your account under "My Templates".


Save my template
The Builder is automatically saving all your changes.

If you have any further questions about our Online Template Builder please email us to support@dynamicxx.com

MailChimp

The following steps will show you how to upload a template into MailChimp and how to use the MailChimp Builder.

1. Before you start with the MailChimp builder I advise you to replace the background images first because you can't change them in the MailChimp system. If you want to change them you need to open the "Images folder" and replace the background image(s). Please upload large background images (2000px width) because Gmail and Yahoo Mail don't accept CSS for background images which means we it won't center and stretch in their system.

2. Compress the HTML file + Images folder.

3. Login into MailChimp or Create an Account.
4. Click on "Templates".

5. Click on "Create Template".

6. Click on "Code Your Own".

7. Click on "Import Zip"

8. Name your Template, choose the Zip file and click on "Upload".

9. Now you can click on "Edit Design" at the bottom of the page to edit the template in the MailChimp builder but I advice to click on "Save and Exit" to move further to the next steps because you can't hide modules in this MailChimp editor. In the next steps I'll explain how you can use the MailChimp builder with all features.


10. If you've clicked on "Save and Exit" you'll see you are in the beginning of the MailChimp system. Now you need to click on "Campaigns".

11. Click on "Create Campaign".
12. Select a Campaign

13. Create a list or Send it to yourself to test it.
14. Name your Campaign + Email Subject and click on next at the bottom of the page.

15. Click on "Saved Templates".

16. Select your saved Template.

17. Now you can edit all images and text.

18. Click on the Eye Icon to hide modules. Click on the Plus Icon to duplicate modules. Click Minus Icon to

19. At the top right you can click on "Preview and Test" to test it and see how your Email Template looks.

Campaign Monitor

The following steps will show you how to upload a template into MailChimp and how to use the MailChimp Builder.

1. Before you start with the Campaign Monitor builder I advise you to replace the background images first because you can't change them in the Campaign Monitor system. If you want to change them you need to open the "Images folder" and replace the background image(s). Please upload large background images (2000px width) because Gmail and Yahoo Mail don't accept CSS for background images which means we it won't center and stretch in their system.

2. Compress the HTML file + Images folder.

3. Login into Campaign Monitor or Create an Account.
4. Click on your avatar and click on "Templates".

5. Click on "Import".

6. Name your Template, Choose HTML file, Choose .Zip file and click on "Add Template".

7. Click on "Campaigns".

8. Click on "Create a New Campaign".

9. Name your Campaign and click on "Next".
10. Click on your saved Template.
11. On the right side of the modules you can click on Edit, Delete, Add New to modify the module.

12. To edit the image itself, you just need to click the cog and then use the Remove Image option to delete the existing image. You can then import a new one by clicking on the "Choose file" button.
